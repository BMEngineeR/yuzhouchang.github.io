## 🔧 Customization Options

### Easy Color Changes
Edit variables in `_sass/_light-theme.scss`:
```css
:root {
  --accent-blue: #4facfe;
  --accent-purple: #667eea;
  --light-bg: #f8f9fa;
}
```

### Photo Styling
Modify photo dimensions and effects:
```css
.showcase-photo {
  width: 200px; /* Adjust size */
  height: 200px;
  border: 5px solid var(--accent-blue);
}
```
